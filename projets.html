Création d'un jeu VR éducatif (HistoryForChildren)

Le but de ce projet était de créer un jeu en VR éducatif pour des enfants dans le cadre de mes études.

Ce projet de deux mois, contient un monde préhistorique dans le quel le jouer peut se balader dans un sentier où se trouvent des dinosaures.

Des informations sur les dinosaures sont disponibles prêts d'eux, donnant un aspect ludique et éducatif pour les enfants.

Deux autres époques sont en préparation, l'Antiquité ainsi que le Moyen-Âge.

MushRoomPie

Ce projet est un site web contenant sa propre API.

Son API est une API permettant de centraliser des appels aux API des musée enregistrés afin de constituer une base de donnée pour une IA.

Une application développée en Python est aussi disponible afin de télécharger la base de donnée sur son ordinateur.

SAE Morpion en réseau C

Le but de ce projet était de développer un Morpion en C et de l'utiliser en réseau. L'utilisation d'un serveur et d'un client était donc nécessaire. Nous avons donc dû découvrir et utiliser les sockets. La difficulté de ce projet aura été le manque de temps, en effet, nous avions seulement 3 jours pour réaliser ce projet.
